Productivity HUB

A modular productivity system combining time management, habit tracking, skill progression, and task organization into one unified platform.

🚀 Features
⏱ Time Distributor

Breaks your daily time into prioritized focus blocks using percentage-based allocation.

📊 Skill Tracker

Tracks skill development over time and visualizes progression.

🔁 Habit Tracker

A habit system with leveling mechanics and progression multipliers to reinforce consistency.

📋 Task Manager

A flexible task organization system with reusable templates for recurring tasks.
